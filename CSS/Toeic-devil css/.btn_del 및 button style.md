.btn_del 및 button 스타일
===
```
.btn_del {
  margin-left: 10px;
  color: #fff;
  background-color: firebrick;
}

button {
  padding: 10px;
  font-weight: bold;
  font-size: 18px;
  cursor: pointer;
  border: 0 none;
  border-radius: 6px;
  padding: 10px 20px;
  color: #fff;
  background-color: dodgerblue;
}

```
- .btn_del: .btn_del 클래스의 버튼은 배경색을 빨간색으로 설정하여 삭제 버튼처럼 보이게 한다.
- button: 버튼은 두껍고 큰 텍스트, 둥근 모서리, 파란색 배경, 흰색 텍스트, 그리고 커서 변경을 통해 버튼 스타일을 적용한다.
